Drug Search

Drug IQ Detailed Instructions

  1. Start with a single drug. Enter that drug name in the text box. Autocomplete can help if you are unsure about spelling. You can enter the drugs generic name or brand name.
  2. If applicable, enter a second drug to discover adverse reaction reports for multiple drugs administered together.
  3. During this process, a report showing symptoms associated with reported adverse reactions are displayed sorted by most common adverse reaction.
  4. Also displayed, is a listing of detailed reports for each interaction report for a given patient. Click “View Report” to view this detailed information.
